# Student Performance Prediction - EDA

## Overview
This project performs Exploratory Data Analysis on a student dataset to understand factors affecting student performance.

## Dataset
File: student_performance_prediction.csv  
Includes study hours, previous grades, extracurricular activities, and pass/fail status.

## Steps
- Data cleaning: handled missing values
- Data preprocessing: converted categorical data
- Data visualization: created multiple graphs

## Observations
- Students who study more hours are more likely to pass
- Most students have mid-range grades
- Higher study hours lead to better performance
- Many students participate in extracurricular activities
- Overall pass and fail distribution is identified

## Conclusion
Student performance is mainly influenced by study hours, previous grades, and overall engagement.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook
